Drupal Typography

ABOUT

This Drupal 7 module demostrates the style and appearance of content. To be used only to view how common elements are rendered by Drupal.

INSTALLING

- Extract the module into your sites/all/modules directory so it looks like sites/all/modules/typography
- Navigate to Administration -> Modules and enable the module.

USAGE

- Browse to path_to_your_site/typography
- Will render the active theme specific styles
